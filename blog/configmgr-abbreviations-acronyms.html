Configuration Manager Abbreviations & Acronyms

Published on November 14, 2025 • 3 min read

A comprehensive alphabetical list of common abbreviations and acronyms used in Microsoft Configuration Manager (formerly SCCM/MECM).

The List

  • ADR - Automatic Deployment Rule
  • AI - Asset Intelligence
  • CB - Current Branch
  • CCM - Configuration Manager Client
  • CCMCache - Configuration Manager Client Cache
  • CCMSetup - Configuration Manager Client Setup
  • CI - Configuration Item
  • CM - Configuration Manager
  • CMG - Cloud Management Gateway
  • ConfigMgr - Configuration Manager
  • DP - Distribution Point
  • EP - Endpoint Protection
  • FSP - Fallback Status Point
  • MECM - Microsoft Endpoint Configuration Manager
  • MCM - Microsoft Configuration Manager
  • MP - Management Point
  • OOB - Out-of-Band Management
  • OSD - Operating System Deployment
  • PKI - Public Key Infrastructure
  • PXE - Preboot Execution Environment
  • RBAC - Role-Based Access Control
  • SCCM - System Center Configuration Manager
  • SMP - State Migration Point
  • SMS - Systems Management Server (legacy)
  • SRS - Software Reporting Services (legacy)
  • SSRS - SQL Server Reporting Services
  • SUG - Software Update Group
  • SUM - Software Update Management
  • SUP - Software Update Point
  • WDS - Windows Deployment Services
  • WIM - Windows Imaging Format
  • WMI - Windows Management Instrumentation
  • WOL - Wake-on-LAN
  • WQL - WMI Query Language
  • WSUS - Windows Server Update Services

Note: This list is continuously evolving. Missing an acronym? Let me know at maskiba@skibatech.com
